# MEMORY GAME

## About the Project

This project is a PvP Memory Game where players compete to find matching pairs of cards. The game allows customization of the number of cards and the score required to win, featuring an interactive menu, a minimalist design, and a light/dark mode.

## How It Works

The game consists of a board with hidden cards. Players take turns revealing two cards at a time.

Available features include:

* Custom Number of Cards

  * Change the number of cards used in the match

* Custom Score

  * Set the score required to win

* PvP Gameplay

  * Two players compete to find matching pairs

* Interactive Menu

  * Configure the game before starting

* Light/Dark Mode

  * Switch between light and dark themes

* Minimalist Design

  * Simple and intuitive interface

```bash
cd memory-game
```

Open the application

```bash
index.html
```

or simply double-click the file.

## How to Use

Configure the game settings using the menu.

Start the match and take turns revealing two cards.

If the cards match, the player scores a point. Otherwise, the turn passes to the other player.

The first player to reach the selected score wins the game.
